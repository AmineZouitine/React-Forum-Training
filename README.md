## ​⚒️​ How to use it 

```sh
git clone git@github.com:AmineZouitine/React-Forum-Training.git
cd React-Forum-Training
npm install
```
Inside de .env file, add you mangoDB database link

In your first terminal
```sh
npm start 
```
In the second 
```sh
npm run serv
```
